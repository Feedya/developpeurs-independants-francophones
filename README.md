# developpeurs-independants-francophones
Liste collaborative des projets lancés par des développeurs indépendants francophones.

# Guide de contribution

Merci de vouloir ajouter ton projet à la liste des développeurs indépendants francophones !  
Pour garder une base propre et utile à tous, merci de respecter les quelques règles ci-dessous.

##  Comment ajouter un projet

Deux méthodes sont possibles :

1. **Faire une Pull Request** (recommandé)  
   - Fork le dépôt, ajoute ton projet dans le fichier `README.md` à la bonne date, puis ouvre une PR.
2. **Ouvrir une Issue**  
   - Donne le nom du projet, le lien, ton pseudo GitHub et une courte description.  
   - Nous nous chargerons de l’ajouter.

##  Critères d’éligibilité

Seuls les produits **ouverts au public** sont acceptés :

- Doivent être un **site web** ou une **application mobile / desktop** (iOS, Android, macOS, Windows, Linux, etc.).
- Ne sont pas acceptés : les outils réservés aux développeurs (IDE, bibliothèques, CLI sans interface utilisateur finale) ni les forums.
- Le projet doit être **fonctionnel** (même en bêta). Les simples pages “coming soon” sont refusées.

##  État du projet

Chaque projet possède un statut, à choisir parmi les trois icônes suivantes :

| Statut | Signification |
|--------|---------------|
| 🕗 | En développement |
| ✅ | Lancé, accessible au public |
| ❌ | Fermé ou plus maintenu |

Si ton projet évolue, tu peux mettre à jour l’icône à tout moment par une nouvelle PR.

## 🖊️ Rédaction de la description

Une bonne description donne envie de tester le produit en une phrase.

**Exemples :**

- ❌ `Outil de génération vidéo par IA`
- ✅ `Plus besoin d’être monteur : créez une vidéo verticale en 3 étapes, génération automatique des sous-titres et des transitions`


##  Format d’entrée

Les projets sont regroupés par **date d’ajout**. Ajoute le tien en respectant rigoureusement ce format :

Exemple :
2026-07-14 ajout
JeanDupont - GitHub
✅ MonAppMétéo : une app météo minimaliste qui affiche la pluie à l'heure près, sans pub - site web


- Si tu n’as pas de compte GitHub, mets simplement un tiret ou laisse vide.
- Le lien vers ton dépôt ou site est placé directement dans le nom du produit (comme dans la liste existante).

##  Processus de revue

- Chaque ajout est vérifié **manuellement** par un mainteneur sous 24 à 48h.
- Si des corrections mineures sont nécessaires (description trop vague, format incorrect), nous le ferons nous-mêmes.
- En cas de refus, nous t’expliquerons la raison dans le fil de discussion.

##  Langue

Le dépôt est rédigé en **français**. Les descriptions des projets peuvent être dans la langue originale du produit, mais un court résumé en français est apprécié.

##  Des questions ?

Ouvre une Issue, nous sommes là pour t’aider !  
Ce dépôt est géré bénévolement – un peu de patience et de bienveillance fera toujours plaisir. 🙏
